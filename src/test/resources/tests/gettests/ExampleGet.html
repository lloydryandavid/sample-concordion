Example GET Test via RESTful Web Service

Description

The goal of this test is to provide a demonstration on how WireMock, and Concordion Test Framework can be used for testing WebServices that implements the architectural style of RESTful.


GIVEN the Web Service is up and running.


WHEN a call to the GET method was made using the URI: "http://localhost:1080/greeting"


THEN the response code should be 200


AND the response body should be Hello World!
